About Me

Saya adalah mahasiswa Teknologi Informasi semester 7 di Universitas Teuku Umar. Saya memiliki ketertarikan besar pada bidang pengembangan web, database, dan teknologi modern. Dengan pengalaman dalam berbagai bahasa pemrograman dan framework, saya terus berusaha mengasah kemampuan teknis maupun soft skill seperti komunikasi, kerja sama tim, dan problem solving. Saya percaya bahwa pembelajaran adalah proses tanpa akhir, sehingga saya selalu terbuka terhadap tantangan baru untuk berkembang dan berkontribusi di dunia teknologi informasi.

Di luar dunia teknologi, saya juga memiliki hobi yang cukup beragam. Saya menyukai makeup sebagai media untuk mengekspresikan diri dan kreativitas. Selain itu, saya senang menari, karena bagi saya tarian bukan hanya gerakan, tetapi juga cara menyalurkan energi positif dan rasa bahagia. Saya juga gemar mencoba hal-hal baru, baik itu pengalaman, keterampilan, maupun tantangan, karena saya percaya setiap hal baru selalu memberi pelajaran berharga untuk berkembang.
